  <name>Task 1: 实现带防抖的文件监控器</name>
  <files>src/lib/config.js</files>
  <action>
    重构 watchConfig 和 stopWatching 函数以实现以下功能：

    1. **防抖机制**：
       - 创建 debounceTimer 变量存储定时器
       - 在 handleChange 函数中清除之前的定时器
       - 设置新的定时器（500ms 延迟）
       - 只在定时器到期时执行重载

    2. **平台兼容性**：
       - 监听所有事件类型（change、rename）
       - macOS 可能触发 'rename' 事件，Linux/Windows 通常触发 'change'
       - 不依赖事件类型判断，依赖防抖机制

    3. **错误恢复**：
       - 重载失败时保持旧配置有效
       - 记录详细错误信息（不暴露敏感信息）
       - 不抛出异常，允许应用继续运行

    4. **事件通知**：
       - 重载成功时调用 onChange(newConfig)
       - 传递新配置给监听器
       - 监听器可以更新应用状态

    5. **清理资源**：
       - stopWatching 关闭文件监控器
       - stopWatching 清除防抖定时器
       - 防止内存泄漏

    **实现模式**（来自 RESEARCH.md）：
    ```javascript
    let configWatcher = null;
    let debounceTimer = null;

    export function watchConfig(onChange) {
      if (configWatcher) {
        configWatcher.close();
      }

      configWatcher = fs.watch(CONFIG_PATH, (eventType, filename) => {
        // macOS 可能触发 'rename'，Linux/Windows 通常触发 'change'
        if (eventType !== 'change' && eventType !== 'rename') {
          return;
        }

        // 防抖：清除之前的定时器
        if (debounceTimer) {
          clearTimeout(debounceTimer);
        }

        // 设置新的定时器
        debounceTimer = setTimeout(async () => {
          try {
            const oldConfig = config;
            const newConfig = await loadConfig();
            console.log('[github-connector] Configuration reloaded successfully');
            if (onChange) {
              onChange(newConfig, oldConfig);
            }
          } catch (error) {
            console.error('[github-connector] Failed to reload configuration:', error.message);
            // 保持旧配置有效
          }
          debounceTimer = null;
        }, 500);
      });

      console.log(`[github-connector] Watching config file: ${CONFIG_PATH}`);
    }

    export function stopWatching() {
      if (configWatcher) {
        configWatcher.close();
        configWatcher = null;
        console.log('[github-connector] Stopped watching config file');
      }
      if (debounceTimer) {
        clearTimeout(debounceTimer);
        debounceTimer = null;
      }
    }
    ```

    **不要**：
    - 不要在重载失败时更新 config 变量
    - 不要移除现有的 watchConfig 和 stopWatching 函数签名
    - 不要硬编码防抖延迟（考虑未来可配置）
